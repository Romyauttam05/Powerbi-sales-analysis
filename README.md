Project Overview

This project focuses on analyzing retail store sales data to understand sales performance, customer purchasing patterns, and product trends. The dashboard provides clear and interactive visual insights to support data-driven business decisions.

Objectives

  Analyze overall sales performance of the retail store
  Identify top-performing products and categories
  Understand sales trends across time periods
  Provide actionable insights for improving business performance

Dataset

  Retail store sales data (Excel/CSV format)
  Includes information such as sales, quantity, product category, and dates

Tools & Technologies

    Power BI Desktop
    Microsoft Excel
    Power Query
    Basic DAX

Key Features

  Interactive dashboard with slicers and filters
  KPI cards for total sales and performance tracking
  Category-wise and time-based sales analysis
  Clean and user-friendly data visualizations

Key Insights

  Identified best-selling products and high-revenue categories
  Observed monthly sales trends and seasonal patterns
  Highlighted areas of strong and weak sales performance

Conclusion

This project demonstrates my ability to clean, analyze, and visualize data using Power BI. It highlights my skills in transforming raw data into meaningful insights for business decision-making.
